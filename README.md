## üìÇ Project Architecture

The codebase follows a modular and feature-driven architecture for scalability:

- `src/features/currency-converter/`: Core logic, state management, and theme-specific styles.
- `src/components/common/`: Atomic and reusable UI components.
- `src/data/`: Comprehensive dataset for 193+ world countries and currencies.
- `src/services/`: Abstracted data-fetching layer for exchange rates.
- `src/styles/`: Global design tokens and CSS variables.

## üö¶ Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Kashif-Khokhar/Currency-Converter.git
    cd Currency-Converter
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```
